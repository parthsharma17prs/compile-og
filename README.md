# CompliLedger

A blockchain-native platform that automates regulatory compliance using Zero-Knowledge Proofs, Agentic AI, OSCAL, and SBOM.

## Tech Stack

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Getting Started

Prerequisites:
- Node.js and npm (recommend installing via nvm)

Steps:
```sh
# 1) Install dependencies
npm install

# 2) Start development server
npm run dev

# 3) Build for production
npm run build

# 4) Preview production build
npm run preview
```

## Project Structure

- `src/` Application source
- `src/pages/` Routes and pages
- `src/components/` Reusable components
- `public/` Static assets

## Scripts

- `npm run dev` – Start dev server
- `npm run build` – Production build
- `npm run preview` – Preview build locally
- `npm run lint` – Lint codebase

## Customization

- Update site metadata in `index.html`
- Update Tailwind config in `tailwind.config.ts`
- Add new UI components under `src/components/`
